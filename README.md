
**Digital Tools Buying Website**

## Description

This project is a modern and responsive digital tools marketplace built using React. Users can explore various digital products, view detailed features, and add items to their cart. The application provides a smooth user experience with dynamic cart functionality, interactive UI components, and real-time notifications.

## Technologies Used

1. React.js
2. Tailwind CSS
3. DaisyUI
4. JavaScript (ES6+)
5. React-Toastify
6. JSON (for product data)

## Key Features

1. **Dynamic Cart System** – Users can add products to the cart, remove items, and proceed to checkout with real-time updates.
2. **Interactive Notifications** – Toast messages (using React-Toastify) for actions like add to cart, remove item, and checkout.
3.  **Fully Responsive Design** – Optimized for mobile, tablet, and desktop devices with clean and user-friendly UI.
